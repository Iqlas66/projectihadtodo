# 🌸 Seasonal Card Switcher

Look, it’s a basic course project. It was supposed to be a simple HTML/JS assignment, but I overcomplicated it by throwing **React** and **Framer Motion** at it. 

Is it revolutionary? No. It has 3 buttons and a card. But the UI looks nice, so whatever.

---

## 🛑 Disclaimers

* **Language:** It’s written 100% in **Ukrainian**. Don't expect a translation, I'm not adding localization for a 3-button app.
* **Status:** It's at version 1.0, which means it is finished forever. I am not adding a billions of  updates and  maintainence TO A FRICKING SEASON APP.
* **Animations:** They're a bit janky depending on your mood, but Framer Motion was involved.

---

## 🛠️ Tech Stack

* **React** (because vanilla JS felt too simple)
* **Framer Motion** (for the animations that I may or may not fully like)
* **CSS** ( rounded corners go brrr)
* **Emoji Regex** ( i hate emojis)

## 🚀 How to Run It

If for some obscure reason you actually want to run this locally:

1. Clone the repo:
   `git clone https://github.com/Iqlas66/projectihadtodo.git`
2. Cd into directory( DVD's will never be forgotten): 
   `cd projectihadtodo`
3. Install 5 gigabites of dependencies i put there for no reason:
   `npm install`
4. Run the dev server (i use arch  btw) :
   `npm run dev`
